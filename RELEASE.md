# Release Notes

## v1.1.0 - UI Polish & Onboarding Update

### Highlights
- **Joyride Onboarding System**: First-time players now experience a visually dynamic, 6-step guided "Spotlight" tour to seamlessly explain game settings.
- **Enhanced Theming Selection**: Players can seamlessly swap between 15 total themes using the new Carousel format or an expansive "See All" grid matrix.
- **4 New Thematic Experiences added**: Wizard School, Pirate Cove, Ninja Dojo, and Farm Life!
- **Non-intrusive Social Call-to-Action**: A gentle pulsing bird pops up on the homescreen containing a follow and star button linked directly to the developer's Github.
- **Audio Improvements**: Responsive bird chirp WebAudio triggered in sync to UI widgets.

## v1.0.0 - Initial Public Release


### Highlights
- Educational integers game inspired by NCERT Class 6 Chapter 10
- Multiplayer mode (2-5 players)
- Vs AI mode
- Difficulty modes: Easy, Hard, Timer
- Animated themes with audio-reactive effects
- Child-focused learning section for integer basics
- Share-ready winner flow

- Refactored into modular React architecture
- Improved board readability for snakes/ladders direction
- Added clearer icon treatment for snake mouth/tail and collisions
- Improved timer behavior and turn handling
- Hint handling fixed and clamped to valid values
