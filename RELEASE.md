# Release Notes

## v1.0.0 - Initial Public Release

### Highlights
- Educational integers game inspired by NCERT Class 6 Chapter 10
- Multiplayer mode (2-5 players)
- Vs AI mode
- Difficulty modes: Easy, Hard, Timer
- Animated themes with audio-reactive effects
- Child-focused learning section for integer basics
- Share-ready winner flow

### Product polish
- Refactored into modular React architecture
- Improved board readability for snakes/ladders direction
- Added clearer icon treatment for snake mouth/tail and collisions
- Improved timer behavior and turn handling
- Hint handling fixed and clamped to valid values
