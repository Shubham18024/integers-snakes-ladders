# Contributing to Integers: Snakes & Ladders

Thanks for contributing 🙌

## Core principles

- Keep UI simple and child-friendly
- Prefer readability over clever code
- Protect learning experience first

## Report bugs

Open an issue with:

- What happened
- Expected behavior
- Steps to reproduce

## Feature requests

Open an issue and explain:

- What you want to add
- Why it helps students

## Suggested contribution areas

- New themes
- New sound packs
- Better accessibility
- Better educational content

## Pull request flow

1. Fork repository
2. Create feature branch
3. Make changes
4. Test locally
5. Open PR with clear description
